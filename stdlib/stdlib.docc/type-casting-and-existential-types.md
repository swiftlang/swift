# Type Casting and Existential Types

Perform casts between types or represent values of any type.

## Topics

### Integer Value Casting

- ``Swift/numericCast(_:)``

### Closure Casting

- ``Swift/withoutActuallyEscaping(_:do:)``

### Instance Casting

- ``Swift/unsafeDowncast(_:to:)``
- ``Swift/unsafeBitCast(_:to:)``

### Existential Types

Use a variable or constant with an existential type to hold an instance of any type.

- ``Swift/AnyObject``
- ``Swift/AnyClass``

### Comparing Identity

- ``Swift/===(_:_:)``
- ``Swift/!==(_:_:)``

### Void Type

- ``Swift/Void``
